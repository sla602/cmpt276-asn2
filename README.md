## Staff Rating Application

### Features
- Create staff ratings
- View staff rating details
- Edit and delete ratings

### Tech Stack
- Spring Boot
- Thymeleaf
- JPA / PostgreSQL
- Maven
- Docker

### How to Run Locally
./mvnw spring-boot:run

### How to Run Tests
./mvnw test

## Deployment
Deployed on Render using PostgreSQL with environment variables.
